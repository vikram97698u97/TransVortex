# Security Policy

## Supported Versions

Only the latest version of TransVortex is supported for security updates.

| Version | Supported |
| ------- | --------- |
| 1.x.x   | ✅ Yes    |
| < 1.0.0 | ❌ No     |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please **DO NOT** open a public issue. Instead, report it privately to the maintainers:

1. Send an email to **security@transvortex.online** describing the vulnerability.
2. Include as much detail as possible, such as a proof of concept (PoC), steps to reproduce, and potential impact.
3. We will acknowledge receipt of your report within 48 hours and coordinate a fix.
4. We aim to resolve all reported vulnerabilities within 30 days of receipt.

Thank you for helping keep this project secure!
